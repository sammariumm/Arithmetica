# Arithmetica: an RPG-like Math Game

A game that makes Math a little more enjoyable

<ins>**Game Mechanics:**</ins>

- An equation is shown to the player
- Each monster contains a name tag containing an answer in this format: `<number>`
- Player must kill the monster containing the correct answer to score points
